DTU Python Development Environment

Overview

The DTU Python Development Environment provides a complete, self-contained Python development setup designed specifically for DTU students and coursework. This hybrid installer combines a constructor-based Python environment with a pre-configured Visual Studio Code installation.

What Gets Installed

Python Environment

  • Python 3.11 - Latest stable Python with full scientific computing stack
  • Conda Package Manager - For easy environment and package management
  • Scientific Packages:
    • pandas - Data manipulation and analysis
    • scipy - Scientific computing library
    • statsmodels - Statistical modeling
    • uncertainties - Uncertainty calculations
    • dtumathtools - DTU-specific mathematical tools

Development Environment

  • Visual Studio Code - Professional code editor
  • Python Extensions:
    • Python extension pack with IntelliSense and debugging
    • Jupyter notebook support
    • PDF viewer for documentation
  • Pre-configured Settings - Optimized for Python development

Installation Locations

Python Environment:

~/dtu-python-stack/

The complete Python environment is installed in your home directory, avoiding system-wide changes.

Visual Studio Code:

/Applications/Visual Studio Code.app

VS Code is installed in the standard Applications folder.

Command Line Tools:

/usr/local/bin/code

The 'code' command is available system-wide for opening files and projects.

Key Features

No Homebrew Dependency

This installer is completely self-contained and does not require Homebrew, eliminating common installation issues and conflicts.

Offline Installation

All Python packages are bundled in the installer, allowing for installation without internet connectivity (VS Code extensions may require internet for initial setup).

Professional PKG Format

Uses native macOS installer technology for reliable, enterprise-ready deployment.

Consistent Environment

Every installation creates exactly the same environment, ensuring reproducible results across different machines.

Troubleshooting

Python Not Found

If Python is not found in your PATH, use the direct path:

~/dtu-python-stack/bin/python3

VS Code Extensions Not Loading

Extensions are installed automatically but may take a moment to activate. Restart VS Code if needed.

Package Import Errors

Ensure you're using the correct Python executable from the DTU installation:

which python3 # Should show ~/dtu-python-stack/bin/python3
